Notre JAR fonctionne comme suivant : 

1, Créer un dossier "test" au même niveau où le JAR se trouve, on met les dossiers à tester dans un specifique dossier "test" dans le même dossier que JAR, et lancer le JAR. 

2, Programme va traiter tous les document .java dans les dossiers et les sous-dossiers puis sauvgarder 4 fichiers csv :


"classes.csv"

"methodes.csv"

"3_BC_classes.csv"  

"3_BC_methods.csv"   

Les deux derniers contiennent "les 3 classes et les 3 méthodes les moins bien commentées et proposez des améliorations".
